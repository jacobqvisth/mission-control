🦞 OpenClaw Setup
Quickstart

Set up OpenClaw in minutes

A clean, practical onboarding page for new users: install the CLI, start the gateway, connect channels, and verify everything works.

Start setup Docs

1) Quickstart checklist

  1. Install OpenClaw CLI
  2. Run the setup wizard
  3. Start the gateway
  4. Check status and health
  5. Connect your first channel (Telegram/WhatsApp/etc.)

2) Install

Use your preferred package manager and verify the CLI is available:

# Example: check available commands
openclaw help

# Verify installation
openclaw --version

If install steps change, always follow the official docs.

3) First-time configuration

# Guided setup
openclaw setup

# Or full interactive configuration
openclaw configure

These commands help configure auth, model, channels, and workspace.

4) Start the gateway service

openclaw gateway start
openclaw gateway status

The gateway is the runtime that powers messaging, tool use, and sessions.

5) Health checks

openclaw status
openclaw doctor
openclaw logs --follow

Use these to confirm channels and agent runtime are healthy.

6) Connect channels

Pick one channel to start with and test message flow:

  • Telegram: configure bot token, set group/mention policy
  • WhatsApp: link personal account and verify session state
  • Others: use openclaw channels --help
openclaw channels --help
openclaw status --deep

Common troubleshooting

Gateway not reachable

Check service status and restart:

openclaw gateway status
openclaw gateway restart
Channel messages not flowing

Run deep status and inspect policy settings (allowlist/mention rules).

openclaw status --deep
Need command reference

Use CLI help and docs:

openclaw help
openclaw docs
