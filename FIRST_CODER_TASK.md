# First Coder Task (Lovable-style)

Project name: lovable-mission-control

Goal:
Build a polished Mission Control web app that feels premium and fast.

Target users:
Jacob (owner/operator)

Core features:
1. Dashboard with key metrics cards
2. Command Center with department agent cards + model selector
3. Projects Kanban board
4. Notes with autosave
5. Important inbox list panel

Design vibe:
Dark, glassmorphism, neon accent, clean spacing, modern SaaS.

Data/auth needs:
- localStorage for state
- optional local server integration at http://localhost:8899

Definition of done:
- Runs locally
- Mobile responsive
- No console errors
- Smooth interactions
- Clear README run steps

Constraints:
- Don’t deploy anywhere
- Don’t touch unrelated files
- Keep code organized and commented
